# Changelog

## 1.0.0

- Initial release
- News feed with search, city filter, and trending stories
- Story detail with like, confirm, share, and report
- Story submission with image upload and AI processing
- Editorial queue and review workflow
- Creator profiles and community features
- KYC verification flow
- Admin dashboard with reports and user management
- Push notification support
- Multi-language support (English, Hindi)
- Pluggable design system and theming
- Auth-optional architecture: viewing is public, actions gate via `AuthProvider.requestLogin()`
